Troubleshooting Instructions

  1. Click the "Access Trading Agent" button above.
  2. If you're redirected back to the dashboard, try opening the Trading Agent in a new tab by right-clicking the button and selecting "Open in New Tab".
  3. If that doesn't work, check if there's any error in the browser's console (F12) when trying to access the page.
  4. Try using the "Access Test Trading Agent" link, which uses a simplified test component.
